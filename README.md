# Multiplayer Poker Game in Java

A Java project implementing a multiplayer poker game with core game logic, hand evaluation, and automated tests.

## Overview

This project was built as a larger programming project in Java.  
The main goal was to design and implement a playable poker game with clean object-oriented structure, correct game rules, and reliable winner evaluation.

The project focuses mainly on:
- poker game logic
- hand evaluation and winner determination
- object-oriented design in Java
- testing and iterative bug fixing

## Features

- Multiplayer poker game logic
- Card dealing and table flow
- Hand comparison and winner determination
- Improved winner message output
- Automated tests
- Maven-based project structure

## Tech Stack

- Java
- Maven
- JUnit

## Project Structure

- `src/` – main source code
- `.mvn/` – Maven wrapper files
- `pom.xml` – project configuration and dependencies

## What I worked on

In this project, I worked on implementing and improving the game logic, fixing bugs, optimizing the code, and making the winner evaluation clearer and more reliable.
